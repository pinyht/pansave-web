帮助中心 ​

PanSave 是一站式网盘媒体自动化管理平台。你可以把它理解成一个长期运行的自部署控制台：先接好 Telegram、网盘、TMDB、Emby，再让它按你的规则扫描频道、追踪榜单、处理豆瓣片单和检查缺集。

如果是第一次部署，建议先看 快速开始，再按自己需要的功能阅读下面的帮助。

使用顺序 ​

  1. 部署并访问 http://localhost:16868。
  2. 使用默认账号 admin/password 登录，并尽快修改账号或密码。
  3. 填写授权码，授权通过后进入系统。
  4. 配置网盘账号，并在转存策略里指定整理源目录。
  5. 配置 Telegram 账号或公开频道模式，再维护要使用的频道。
  6. 配置 TMDB 和 Emby，启用需要的监控策略。
  7. 按场景使用频道扫描、榜单订阅、豆瓣求片或 Emby 缺集扫描。

几个常见术语 ​

名称简单解释
Telegram API 模式使用你自己的 Telegram 账号读取已加入频道，适合私有频道或需要登录才能访问的频道。
公开频道模式不登录 Telegram，只读取公开可访问的 t.me 频道网页，适合公开频道。
TMDB影视资料库，用来确认电影、剧集、年份、海报和季集结构。
Emby你的媒体库服务，PanSave 用它确认资源是否已经入库、是否缺集。
整理源目录网盘里接收新资源的目标目录。频道扫描、监控、豆瓣求片转存时都会用到它。
监控清单系统正在跟踪的电影或剧集列表，来自榜单、自定义加入、豆瓣求片镜像或缺集补救。
